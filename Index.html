Informasi Pelatihan

Proyek ini bertujuan untuk membekali peserta dalam lingkup pengembangan web melalui penerapan tiga teknologi utama: HTML, CSS, dan JavaScript. Peserta akan diberikan pelatihan dari penerapan struktur dasar HTML, penggunaan CSS untuk menambah styling, memperkenalkan dasar-dasar JavaScript, hingga penggunaan PHP dan SQL untuk mengintegrasikan backend. Di sini, peserta memiliki kesempatan untuk membangun website sederhana yang mengkombinasikan aspek frontend dan backend, memberikan pemahaman yang solid tentang bagaimana komponen-komponen tersebut bekerja dalam pengembangan web.
